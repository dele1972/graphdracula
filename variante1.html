Graph Dracula

Variante 1

Hier habe ich versucht die Graph Dracula JS Lib manuell einzubinden. Die Dokumentation gibt nicht viel her und Beispielscripte mit komplettem Aufbau oder ein standardisierter Aufbau wird nicht mitgegeben.

Als erstes habe ich zusätzlich Raphael JS eingebunden (siehe auch: "An intro to Raphael") - eine Bibliothek, mit welcher per JS eine SVG erstellt werden kann. Diese wird scheinbar benötigt.

Weiterhin wird wohl 'graffle JS' benötigt, welches auch irgendwie mit Raphael in Verbindung steht (ich habe mir das noch nicht näher angesehen).

Wenn man das Beispiel aus GraphDracula verwendet, ist dann trotzdem noch Graph unbekannt. Hier habe ich eine alte Version im Netz gefunden ( SNIPPLR: Graph JavaScript framework, version 0.0.1 ), die allerdings auf anhieb nicht funktionierte. Ohne weiteren Aufwand betreiben zu wollen, habe ich auch eine modifizierte Version von Philipp Strathausen gefunden (in Variante 2 wird eine scheinbar andere Version davon eingesetzt).
